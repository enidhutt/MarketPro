# Рост конверсии маркетплейса MarketPro: от метрик до гипотез

**Моя роль:** Продакт-менеджер (B2C, опыт покупателей)  
**Продукт:** MarketPro — маркетплейс Non-Food (одежда, электроника, товары для дома). Аналог Wildberries/Ozon для среднего ценового сегмента.

**Ключевые метрики продукта на старте:**

| Метрика | Значение |
|---------|----------|
| DAU (пользователи в день) | 10 000 |
| Конверсия в заказы (CR) | 0.5% |
| Заказов в день | 50 |
| Средний чек (AOV) | 3 000 ₽ |
| GMV в день | 150 000 ₽ |

**Запрос CEO:** «Из 10 тысяч человек — 50 заказов?! Где теряются пользователи? Когда будет нормальная выручка?!»

**Проблема:** Конверсия 0.5% (50 заказов из 10 000 пользователей в день)  
**Цель:** Найти точки роста и увеличить GMV с 150 000 ₽ до 250 000 ₽ (+60%)  
**Результат:** Дерево метрик, NSM, полная воронка, 15 гипотез, дашборд в DataLens

---

## Задачи

1. Построить дерево метрик от GMV до продуктовых прокси-метрик - 
[Дерево метрик в Miro (скриншот)](src/metrics/Дерево_метрик_MarketPro.png)

2. Выбрать главную метрику продукта (North Star Metric - NSM), отражающую ценность для клиента -
[Таблица NSM (Google Sheets)](src/metrics/NSM_MarketPro.xlsx)

3. Построить сквозную и относительную воронки от 10k пользователей до 50 покупателей - 
[Таблица с воронками (Google Sheets)](src/funnel/Воронка_MarketPro.xlsx)
[Скриншот диаграмм](src/funnel/Воронка_диаграммы.png)

4. Сформулировать 12+ гипотез для улучшения воронки - 
[Скриншоты промптов Perplexity](src/hypotheses/Perplexity_промпты.png)
[Таблица с 15 гипотезами (Google Sheets)](src/hypotheses/Гипотезы_MarketPro.xlsx)

5. Создать живой дашборд для CEO с ключевыми метриками:
   - Динамика конверсии по дням (с разбивкой по каналам)
   - Динамика GMV по дням
   - Динамика Orders по дням
   - Воронка продаж (Users → Views → Add2Cart → Checkout → Orders)
